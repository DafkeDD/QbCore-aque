Help:
 För att flytta en full stack se till att BELÖPSFÄLTET är tomt. Du kan ange det specifika ANTAL BELOPP för hur mycket du vill flytta. För att ANVÄNDA ett objekt, dra det till mittalternativet med titeln "ANVÄND" eller så kan du DUBBELKLICKA. BELÖPSFÄLTET återställs INTE när du stänger ditt lager, så se till att du tar för vana att rensa numret!
Mellanalternativ:
  • Använd: Dra objektet till den här knappen för att använda objektet
  • Ge: Ge föremål till andra medborgare runt omkring dig
  • Kombinera: Kombinera vissa föremål
  • Switch: Byt aktuellt objekt
  • Bilagor: När du har föremål som du kan fästa på dina vapen
  • Stäng: Ett annat sätt att stänga inventeringen
Keybinds (standard):
  • Vänsterklick: Normalt drag
  • Högerklicka: Släpp objekt eller flytta enstaka objektlagring
  • Dubbelklicka: Använd objekt
  • Nummer 0-9: Matar in det inskrivna numret
  • BACKSPACE-tangent: Kommer att ställas in SUMMA för att tömma/flytta hela objektstapeln
